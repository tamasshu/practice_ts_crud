## 環境構築

### 必要なバージョン

node.js: 20.10.0
Next.js: 14.2.18
styled-components

### インストール手順

1. 必要なパッケージをインストールする

```bash
npm install
```

2. yup と yupResolver と react-hook-form をインストール

```bash
npm install @hookform/resolvers yup
npm install react-hook-form
```

3. 開発サーバーを立ち上げる

```bash
npm run dev
```

4. [http://localhost:3000](http://localhost:3000) で確認する
